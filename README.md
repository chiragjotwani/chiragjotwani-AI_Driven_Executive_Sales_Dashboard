# AI-Driven Executive Sales Dashboard

## Project Overview
This project demonstrates how modern **business intelligence dashboards** can be enhanced using **Large Language Models (LLMs)** to generate executive-level insights automatically.

The solution combines:
- **Power BI** for interactive data visualization
- **Python + LLMs** for natural-language business insights

The dashboard is designed for **non-technical stakeholders** (CXOs, managers) to quickly understand sales performance and trends without manual analysis.

---

## Objectives
- Build an executive-friendly **Power BI sales dashboard**
- Derive meaningful business metrics from a raw dataset
- Use LLMs to generate **automated textual insights**
- Demonstrate an end-to-end analytics workflow suitable for industry use

---

## Tech Stack
- **Power BI** – Data modeling, DAX, dashboards
- **Python** – Data handling and LLM integration
- **LLMs (OpenAI / compatible APIs)** – Insight generation
- **Git & GitHub** – Version control and project tracking

---

## Dataset
The dataset contains historical sales records including:
- Order date
- Region
- Product category
- Sales
- Profit

> Note: Since quantity and discount fields were unavailable, key metrics were derived using calculated columns in Power BI.

---

## Key Features
- KPI cards for Total Sales, Profit, and Profit Margin
- Time-series analysis of sales performance
- Category-wise and region-wise sales breakdown
- Automated executive summaries using LLMs
- Natural language explanation of trends and anomalies
